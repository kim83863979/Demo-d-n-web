II. HƯỚNG DẪN CÀI ĐẶT (SETUP CƠ SỞ DỮ LIỆU)
Để hệ thống hoạt động trơn tru với đầy đủ dữ liệu demo, kính mong thầy thực hiện các bước sau:
1. Copy thư mục source code vào htdocs (nếu dùng XAMPP).
2. Mở phpMyAdmin, tạo một Database mới tên là: store.
3. Import file CSDL bản chuẩn nhất có tên là "store.sql" (nằm ngay trong thư mục này) vào Database vừa tạo.
